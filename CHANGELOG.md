# Changelog

## [1.0.0] - 2025-04-29

### Agregado
- Dashboard de diagnóstico de red LAN/WAN
- Detección automática de IP local, gateway, DNS y adaptador
- Medición de latencia, jitter y pérdida de paquetes
- Diagnóstico inteligente con 7 reglas de correlación LAN vs WAN
- Detección de tipo de conexión (WiFi / Ethernet) con señal en dBm
- Conteo de dispositivos activos via tabla ARP
- Score de calidad de conexión 0-100
- Interfaz en tkinter puro (sin dependencias externas)
- Compatible con entornos corporativos (sin PowerShell, sin scripts)
- Auto-elevación a administrador via UAC
- Compilación automática a .exe via GitHub Actions
